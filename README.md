**Práctica de Simulador de campo eléctrico**

Proyecto con fines de aprendizaje, aplicando conceptos de electricidad como:
- Campo eléctrico
- Cargas puntuales
- Fuerza eléctrica
- Principio de superposición

**Carácteristicas**
- Agregar y eliminar cargas positivas como negativas con cargas puntuales
- Visualización de campo eléctrico producido por cargas puntuales
- Campo eléctrico en tiempo real
